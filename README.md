# SASS Tutorial

A SASS/SCSS tutorial project demonstrating CSS preprocessing techniques including variables, mixins, nesting, and partials.

## Project Structure

- `scss/` — SASS source files
- `css/` — Compiled CSS output
- `img/` — Image assets
- `final/` — Final compiled version

## Getting Started

Use a SASS compiler (e.g., Prepros, node-sass, or dart-sass) to compile the SCSS files.
